## ETL Decisions

### Decision 1 — Date Format Standardization
Problem: The dataset had inconsistent date formats.
Resolution: Converted all dates into standard YYYY-MM-DD format.

### Decision 2 — Handling Missing Values
Problem: Some rows had NULL values in important columns.
Resolution: Removed or replaced NULL values to ensure data quality.

### Decision 3 — Category Standardization
Problem: Product categories had inconsistent naming (e.g., electronics, Electronics).
Resolution: Standardized all category names into proper format (e.g., Electronics).
